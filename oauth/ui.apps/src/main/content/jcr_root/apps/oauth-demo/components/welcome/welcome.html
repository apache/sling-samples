About the demo

This application shows the expected basic usage of the Sling OAuth Client bundle (whiteboard)

It exposes two pages:

  • The Youtube videos page ( oauth-demo/components/youtube-videos), implemented using a Sling Servlet ( YoutubeSearchServlet)
  • The GitHub repositories page ( oauth-demo/components/github-repos), backed by a Sling Model ( GithubRepositoriesModel)

To use the application you will need to register this application as OAuth clients to Google and/or Github.
